Residuals: Residuals are the amount of distance from the data point to the line. Drawing a line from the point to the line is the residual.

Residuals should be normally distributed around 0.

Coefficients: Y-Intercept - conceptually meaningless in this model. Weight vs. Height y-intercept is a number saying ‘when height is 0, weight is x.’ Height can not be 0 so its not important in this data set.

Height - 3.4500 means that when height increases 1, weight increases 3.4500. The p-value is exeptionally small at 37.85 1.09e-14 meaning that we reject the null hypothesis that there is not a relationshiop between height and weight. So this height value is statistically significant.

F-statistic: the p-value is small 1.091e-14 meaning that the f-statistic is statistically significant. Meaning that at least one of the predictor variables is explanatory or is predictive of the outcome variable. So if the F statistic is statistically significant then that is good. If p value is big then the model doesnt work.

R^2: Multiple R-squared: 0.991 means that 99.1% of the change is weigh can be explained by a change in height.

Adjusted R^2: is really only for multiple regression.
